# Volleyball-Turnierverwaltung 2.2.0

## Neu

- Jedes flexible Spielfeld hat jetzt einen eigenen Namen und eine Platzart: Sand, Rasen, Halle oder Sonstiges.
- Mehrere Plätze derselben Art sind möglich, z. B. zwei Rasenplätze.
- Der Generator verteilt die Einsätze je Platzart möglichst gleichmäßig und berücksichtigt dabei die Anzahl der vorhandenen Plätze. Bei zwei Rasenplätzen und je einem Sand- und Hallenplatz wird Rasen entsprechend häufiger genutzt.
- Die Fairnessprüfung zeigt je Team, wie oft es auf jeder Platzart gespielt hat.
- Bei aktivierter Finalrunde kann die genaue Reihenfolge der acht qualifizierten Teams in der öffentlichen Live-Tabelle verborgen werden. Die Teams werden dann nur als „Qualifiziert für die Finalrunde“ angezeigt; im Bearbeitungsmodus bleibt die genaue Tabelle sichtbar.

## Installation

Alle Dateien in das GitHub-Pages-Repository hochladen und bestehende Dateien ersetzen. Eine Änderung an Supabase ist nicht notwendig, da die neuen Einstellungen im vorhandenen JSON-Datensatz gespeichert werden.
